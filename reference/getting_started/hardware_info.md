# Hardware Information

!!! warning "Fairy Type Compatibility Issues"

    The Fairy Type implementation is known to break certain kinds of setups. Check the compatibility lists below before playing.

## Compatibility Status

=== "Working Correctly"

    **Emulators - PC**

    - **DeSmuME** - Fully compatible
    - **MelonDS** - Fully compatible

    **Emulators - Mobile**

    - **MelonDS (Android)** - Fully compatible
    - **Drastic** - Fully compatible
    - **iNDS** - Fully compatible

    **Hardware/Flashcarts**

    - **TWiLight Menu++** (latest version)
        - **Important:** Launch in DS mode, otherwise the game will white screen!
    - **RTS R4i (Gold)+** (3DS/DS) - Fully compatible

=== "Incompatible"

    **Known Issues**

    - **NDS forwarder** - Does not work with Redux

---

!!! question "Found a Compatibility Issue?"

    If you encounter a compatibility issue not listed here, please contact the Redux developers on [Twitter](https://x.com/aphexcubed) or Discord.
